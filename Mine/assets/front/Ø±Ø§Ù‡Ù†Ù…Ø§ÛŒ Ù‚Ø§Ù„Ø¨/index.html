ادغام گوگل ریکپچا در وبسایت خود.

برای اغام آن در وبسایت خود باید API گوگل ریکپچا را در طرف سرویس گیرنده و همچنین سرویس فرستنده قرار دهید

برای نمایش ویجت در فرم خود باید این تکه کد را در زیر فرم تماس، فرم نظر یا هر فرم دیگری قرار دهید و کد سایتی که در بالا از گوگل دریافت کردید را در قسمت Your site key قرار دهید.

وقتی فرم به سرور ارسال شود، اسکریپت "جواب ریکپچا" را به عنوان داده "پست" ارسال می کند. برای اینکه ببینید آیا کاربر کپچا را تایید کرده است یا خیر، باید آن را تأیید کنید.

برای درک بهتر پیشنمایش قالب را ببینید

MailChimp -

MailChimp یک سرویس بازاریابی از طریق ایمیل است، به شما کمک می کند مشتریان وب سایت خود را مدیریت کنید. MailChimp روشی آسان برای ادغام فرم ثبت نام ایمیل در وب سایت و ارسال خبرنامه ایمیل برای مشتریان را فراهم می کند. با استفاده از این طرح، می توانید حداکثر بیش از 2000 مشترک به MailChimp اضافه کنید و ماهانه 12000 ایمیل برای مشتریان خود ارسال کنید.

در این آموزش، به شما نشان خواهم داد که چگونه فرم اشتراک خبرنامه را در وب سایت خود ادغام کنید و مشتریان را با استفاده از PHP به لیست MailChimp اضافه کنید. برای افزودن مشتریان به لیست بدون ایمیل تأیید، از MailChimp API و PHP استفاده خواهیم کرد.

برای ادغام API MailChimp در PHP به یک کلید MailChimp API و List ID نیاز دارید که می خواهید اعضا را به آن اضافه کنید. قبل از شروع، به یک حساب MailChimp نیاز دارید، مراحل زیر را برای دریافت API Key و List ID دنبال کنید.

ساخت API Key و List ID

دریافت API Key:

1. وارد حساب خود در MailChimp شوید.

2. در فهرست کشویی منوی کاربری در سمت راست بالا، روی Account کلیک کنید.

3. از منوی ناوبری بالا Extras را باز کنید و روی API Keys کلیک کنید .

در قسمت پایین Your API keys با کلیک روی Create A Key button یک API Key جدید تولید کنید، پس از تولید آن را برای استفاده های بعدی در PHP کپی کنید.

دریافت List ID:

1. 1. لیستی را انتخاب کنید که تمام اطلاعات مشترک در آن ذخیره شود. در این صورت ، شما باید یک لیست در صفحه لیست ها ایجاد کنید.

وارد لیست ایجاد شده خود شوید و به مسیر Settings » List name and campaign defaults بروید

در قسمت پایین List ID کد List ID خود را پیدا میکنید. آن را برای استفاده های بعدی کپی کنید

در اسکریپت نمونه، یک فرم اشتراک خبرنامه ساده اجرا می شود. هنگامی که کاربر فرم اشتراک را به همراه جزئیات (نام ، نام خانوادگی و ایمیل) ارسال می کند ، مشخصات مربوطه با استفاده از MailChimp API و PHP به لیست حساب MailChimp اضافه می شود. همچنین مشترک می تواند خبرنامه وب سایت شما را از طریق MailChimp دریافت کند.

مشترک را به لیست MailChimp اضافه کنید (mailchamp.php)

در فایل mailchamp.php ، داده های فرم اشتراک دریافت می شود و جزئیات مشترک را با استفاده از MailChimp API و PHP به MailChimp ارسال می کنید. برای استفاده از MailChimp API ، باید API Key و List ID را وارد کنید. کلید API و شناسه لیست را که در مرحله قبل دریافت کرده اید وارد کنید.

برای افزودن یک مشترک ، باید وضعیت مشترک را در فایل JSON خود وارد کنید. اگر وضعیت مشترک را عضو شده کنید، آدرس ایمیل مشترک بدون ارسال ایمیل تأیید بلافاصله اضافه می شود. برای ارسال ایمیل تأیید به مشترک، از وضعیت pending استفاده کنید.

نتیجه گیری

این کد به شما کمک می کند تا از طریق فرم اشتراک سفارشی مشترکین را از وب سایت به لیست mailChimp اضافه کنید. لیست مشترک اضافه شده را می توانید از حساب MilChimp مشاهده کنید. وارد حساب MailChimp خود شوید و به لیست مشترکان بروید، مشترکین تازه اضافه شده را در لیست مشاهده خواهید کرد.
